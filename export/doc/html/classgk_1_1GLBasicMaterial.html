Detailed Description

genere un shader program associe a une description de matiere, cf MeshMaterial.

utilise un modele d'eclairement simple : ambient / PI + kd * diffuse + ks * specular

ambient, diffus et specular sont des constantes eventuellement modulees par une texture : diffuse= diffuse_color * diffuse_texture * diffuse_emission * cos specular= specular_color * specular_texture * specular_emission * cos * pow ambient= ambient_color * ambiant_texture * ambiant_emission * abs(cos)

pow utilise le modele de blinn, pow(ns, dot(n, h)) * (terme de normalisation)

cos est dot(normal, light_direction) limite aux valeurs positives ou la valeur absolue en mode both sides.

cf material_viewer et hair_viewer pour un exemple complet. cf envmap_viewer
