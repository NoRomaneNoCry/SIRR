ImageArray* gk::readImageArray ( const std::string &  format,
const unsigned int  size = -1u 
)

fonction utilitaire, importee de ImageIO. charge une sequence d'images numerotees de 1 a n et construit un ImageArray. les images chargees sont mises en cache.

exemple :

// charge les images envmap[1..n].bmp
const char *format= "images/envmap_%05d.bmp";
gk::ImageArray *envmap= gk::readImageArray(format);
// parcours les n images chargees
for(unsigned int i= 0; i < envmap->images.size(); i++)
... faire quelquechose avec envmap->images[i]
