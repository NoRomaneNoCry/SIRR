Member Function Documentation

Point gk::PTNTriangle::texcoord ( const float  u,
const float  v 
) const
inline

calcule les coordonnees de texture d'un point a l'interieur du triangle connaissant ses coordonnees barycentriques. convention t(u, v)= (1 - u - v) * ta + u * tb + v * tc. cf Triangle::sampleUniformUV() pour choisir uniformement une position sur le triangle.
